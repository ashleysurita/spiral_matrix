# spiral_matrix

### Leetcode Problem:
https://leetcode.com/problems/spiral-matrix/

```
Given a matrix, return an array representing the traversal of the matrix in spiral order.

Function Signature:
func spiralMatrix(input: [[Int]]) -> [Int]

Examples
[[1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]]

returns
[1, 2 3, 6, 9, 8, 7, 4, 5]
```
